Why Moringa?

Having taught myself how to code using PHP and mysql, i strongly felt i was limiting myself to back-end web development. i therefore needed to know more about front end development which would also be a strong entry point into mobile development. My research on coding schools in East Africa put no doubt in my mind that Moringa was the best. Particularly, it has the kind of learning approach i have been yearning for in all my coding history. After leaving Moringa-prep, i hope to have learnt more about javascript and develope a few application to showcase, join Core and learn mobile developement. and finally finish the program as a developer who can readily and competitively solve computing problems in the real world
